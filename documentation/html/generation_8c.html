adequation_rangee_et_nombres ( int  S[N][N],
int  periph[N][N],
int  nb_groupes_defaut,
int  rangee,
char  type_rangee 
)

Vérifie dans la matrice solution, pour une matrice périphérique, un compteur d'indice et une rangée précise donnés, si le remplissage d'une rangée de type "ligne" correspond aux nombres de la rangée de type "colonne" (et inversement).

Paramètres
SLa matrice solution dans laquelle on vérifie l'adéquation d'une rangée par rapport à une autre
periphLa matrice périphérique qui contient les nombres nécessaires à la vérification
nb_groupes_defautLa taille normale du groupe de nombres de la rangée
rangeeLa rangée de la matrice solution qui va être vérifiée
type_rangeeLa nature de la rangée (ligne ou colonne)
Renvoie
Cinq valeurs possibles : 2 si un groupe possède plus de cases que prévu, 1 si des groupes supplémentaires sont détectés, 0 si les données correspondent, -1 s'il reste des groupes à créer, -2 s'il reste des cases à compléter

gen_solution ( int  S[N][N],
int  C[N][N],
int  L[N][N],
int  largeur 
)

Initialise la matrice solution grâce aux données des matrices périphériques.

Pour gérer le remplissage progressif et les cas d'erreur, un système de numérotation des cases a été mis en place : 0 => case vide non-testée, 1 => case cochée, 2 => case vide déjà testée, 3 => case cochée validée.

Paramètres
SLa matrice solution qui va contenir le motif du puzzle
CLa matrice périphérique des colonnes (situé en haut de la grille du Picross)
LLa matrice périphérique des lignes (situé à gauche de la grille du Picross)
largeurLa taille maximum des rangées pour les matrices périphériques
Renvoie
Ne retourne aucun résultat

verif_adequation_globale ( int  soluce[N][N],
int  C[N][N],
int  L[N][N],
int  respect_C[N],
int  respect_L[N] 
)

Lorsqu'une itération de la boucle principale de la fonction "gen_solution" a été effectuée, cette fonction a pour rôle de vérifier l'état de toutes les rangées.

Elle remplie les tableaux "respect_" avec un indicateur numérique : 0 pour une rangée qui ne respecte pas ses règles, 1 pour un résultat positif.

Paramètres
soluceLa matrice solution
CLa matrice périphérique des colonnes
LLa matrice périphérique des lignes
respect_CLe tableau qui va contenir les résultats du test pour les colonnes des matrices soluce et C
respect_LLe tableau qui va contenir les résultats du test pour les lignes des matrices soluce et L
Renvoie
Ne retourne aucune variable
